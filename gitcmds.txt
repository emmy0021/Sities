Clone repo:			git clone <siteadress>
List branches:			git branch
Switch branch:			git checkout <branchname>
Merge branch:			git merge <branchname>
Create branch:			git branch <branchname>
Delete branch:			git branch -d <branchname>
Add changes (step 1):		git add <filename>
Prepare changes (step 2):	git commit -m <"message">
Push changes (step 3):		git push origin <branch>
Shot commit history:		git log
